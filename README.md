1. Understand and Code Data Structures Using Python:
    lists, nested lists, indexing, iteration, matricies
    learn to code abstraction, classes, and modularization in Python. 
2. Understand ML Fundamentals:
    tensors, matricies, array operations
3. Learn Engineering Workflow
